# Funny v2.2.1 — VM 5× and AI Benchmark

**Date:** 2026-07-07

v2.2.1 closes the v2.1.x performance milestone and ships a real LLM benchmark harness.

## Highlights

- **~7× VM speedup** — exec-only `fib(20)` vs tree-walking interpreter; locals pooling and hot-path dispatch
- **`funny bench ai`** — 50-task compile_ok/compile_err suite with parser + typecheck scoring
- **LLM providers** — OpenAI and Anthropic via env keys; `--mock` echoes prompts for harness checks

## Install

```bash
go install github.com/jiejie-dev/funny/v2/cmd/funny@v2.2.1
```
