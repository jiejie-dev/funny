# Funny v2.2.0 — Interactive REPL

**Date:** 2026-07-07

v2.2.0 delivers the v2.2 roadmap item: an interactive read-eval-print loop for learning and experimentation.

## Highlights

- **`funny repl`** — persistent variables, functions, and structs across inputs
- **Multi-line cells** — indentation blocks with `...` continuation prompt
- **Expression results** — trailing expressions print their value (including block tails)
- **Meta-commands** — `:vars`, `:reset`, `:help`, `:quit`

## Install

```bash
go install github.com/jiejie-dev/funny/v2/cmd/funny@v2.2.0
```

## Quick example

```
funny> let x = 10
funny> x + 5
15
```
