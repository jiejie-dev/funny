# Funny v2.1.5 — Mutable Struct Fields (`mut`)

**Date:** 2026-07-07

v2.1.5 adds explicit `mut` struct fields so you can update selected fields after construction while keeping the rest immutable.

## Highlights

- **`mut count: int`** — mark fields that may be assigned after `Counter(count: 0, ...)`
- **Type-checked** — assigning to a non-`mut` field is `E2010` at compile time
- **Full stack** — parser, type checker, compiler (`SET_FIELD`), VM, evaluator, formatter

## Example

```funny
struct Counter:
    mut count: int
    label: str

let c = Counter(count: 0, label: "hits")
c.count = c.count + 1   # OK
```
