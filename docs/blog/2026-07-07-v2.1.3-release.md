# Funny v2.1.3 — Plan Branch Case-List

**Date:** 2026-07-07

v2.1.3 gives the `branch` plan step kind real semantics: a declarative case-list that dispatches to named steps, with planGraph visualization and type checking.

## Highlights

- **`branch` case-list** — `status == 200 => "success"`, `_ => "fallback"`
- **Plan engine** — runs exactly one target step; branch targets are skipped in linear execution
- **Type checking** — validates target step names (E2111) and duplicate step names (E2110)
- **LSP planGraph** — `branch` edges from dispatcher to target steps

## Install

```bash
go install github.com/jiejie-dev/funny/v2/cmd/funny@v2.1.3
```
