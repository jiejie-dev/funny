# Funny v2.2.3 — DAP and VS Code Integration

**Date:** 2026-07-07

v2.2.3 closes the v2.2 editor experience loop: learn in REPL, write with LSP, debug with DAP.

## Highlights

- **`funny dap`** — Debug Adapter Protocol server for VS Code Run and Debug
- **VS Code** — Start REPL in terminal, Debug Funny File launch config
- **Locals + Stack** — inspect bytecode VM state from the editor

## Install

```bash
go install github.com/jiejie-dev/funny/v2/cmd/funny@v2.2.3
```

## VS Code

1. Install/build the extension from `editors/vscode/`
2. Open a `.fn` file → Run and Debug → **Debug Funny File**
3. Or command palette → **Funny: Start REPL**
