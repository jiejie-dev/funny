# Funny v2.1.0 — F-strings, Formatter, LSP, and Module Imports

**Date:** 2026-07-07

v2.1 closes the biggest gaps between what v2.0 documented and what actually worked on the default VM path, and adds the editor/LLM tooling layer the project was missing.

## Highlights

- **F-strings** — `f"total: {price:.2f}"` works end-to-end
- **`funny fmt`** — real AST formatter, wired into MCP `format` and LSP
- **`funny lsp`** — diagnostics, hover, completion, go-to-definition, rename, plan graph
- **`import "mod.fn"`** — multi-file projects with `pub` symbols
- **Plan engine** — `guard`, `delay`, retry backoff, and step timeouts actually run

## Install

```bash
go install github.com/jiejie-dev/funny/cmd/funny@v2.1.0
```

## What's next

v2.1.x follow-ups: `match` execution, `break`/`continue`, `x in list` on the VM path, and the 5× interpreter performance target. See `RELEASE_NOTES.md` for the full limitation list.
