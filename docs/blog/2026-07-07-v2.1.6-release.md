# Funny v2.1.6 — Bytecode Debugger

**Date:** 2026-07-07

v2.1.6 delivers the v2.1 roadmap debugger: source maps at compile time, breakpoints, and single-step execution on the bytecode VM.

## Highlights

- **Source maps** — every instruction maps to `file:line:col`; JSON export via `funny debug --source-map`
- **`funny debug`** — interactive REPL: `step`, `continue`, `break`, `locals`, `stack`
- **`funny disasm`** — instructions annotated with source locations

## Install

```bash
go install github.com/jiejie-dev/funny/v2/cmd/funny@v2.1.6
```

## Quick example

```bash
funny debug script.fn -b 10
# (dbg) step
# (dbg) locals
# (dbg) continue
```
