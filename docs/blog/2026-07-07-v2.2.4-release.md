# Funny v2.2.4 — Package Manager

**Date:** 2026-07-07

v2.2.4 turns the v2.1.7 package manager prototype into a usable workflow: declare, install, update, and import third-party modules with version constraints.

## Highlights

- **`funny pkg add`** — declare a dependency in `funny.pkg` and install in one step
- **`funny pkg update`** — refresh sources and lockfile checksums
- **Version constraints** — `1.2.3`, `>=1.0.0`, `^1.2.0`, `*` validated at install time
- **LSP** — autocomplete `import "pkg:` package names from the project manifest

## Install

```bash
go install github.com/jiejie-dev/funny/v2/cmd/funny@v2.2.4
```

## Example

```bash
funny pkg add math path:vendor/math.fn --version "^1.0.0"
funny pkg list
funny pkg update math
```

In your `.fn` file, type `import "pkg:` and the LSP suggests `math` and other declared packages.
