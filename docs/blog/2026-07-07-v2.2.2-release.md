# Funny v2.2.2 — REPL Learning Environment

**Date:** 2026-07-07

v2.2.2 completes the v2.2 roadmap: `funny repl` is now a guided learning environment, not just a developer scratchpad.

## Highlights

- **Guided tutorials** — `:lesson 1` walks through `docs/tutorial-*.funny` step by step
- **Type exploration** — `:type 1 + 2`, `:desc x`, `:complete pr`
- **Workspace tools** — `:load script.fn`, `:install` for `funny.pkg`

## Install

```bash
go install github.com/jiejie-dev/funny/v2/cmd/funny@v2.2.2
```

## Quick example

```bash
funny repl --lesson 1
# :step    — run demo code for current step
# :hint    — read the hint
# :type greeting + "!"
```
