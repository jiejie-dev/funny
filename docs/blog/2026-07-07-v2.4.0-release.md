# Funny v2.4.0 — Test Framework and Doc Generator

**Date:** 2026-07-07

v2.4.0 delivers the v2.4 roadmap milestone: native unit tests and API documentation from source.

## Highlights

- **`funny test`** — run `test "name":` blocks in `*_test.fn` files
- **`assert` / `assert_eq`** — builtins for test assertions
- **`funny doc`** — generate Markdown or JSON from `##` doc comments

## Install

```bash
go install github.com/jiejie-dev/funny/v2/cmd/funny@v2.4.0
```

## Example

```funny
## Add two numbers
pub fn add(a: int, b: int) -> int:
    return a + b

test "addition":
    assert(add(1, 2) == 3)
```

```bash
funny test
funny doc . --out docs/api
```
