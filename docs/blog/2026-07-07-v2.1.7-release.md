# Funny v2.1.7 — Package Manager (`funny pkg`)

**Date:** 2026-07-07

v2.1.7 adds the M4 package-manager prototype: declare dependencies in `funny.pkg`, install with checksum locks, and import via `pkg:name`.

## Highlights

- **`funny.pkg`** — JSON manifest with `path:`, `https://`, and `git+` sources
- **`funny.lock`** — SHA-256 checksums for installed packages
- **`import "pkg:math"`** — resolves to `.funny/packages/math/`

## Install

```bash
go install github.com/jiejie-dev/funny/v2/cmd/funny@v2.1.7
```

## Quick example

```bash
funny pkg install
funny run main.fn   # import "pkg:math" works after install
```
