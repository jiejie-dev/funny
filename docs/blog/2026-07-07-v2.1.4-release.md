# Funny v2.1.4 — Typed Errors and retry.on

**Date:** 2026-07-07

v2.1.4 adds runtime error typing for plan steps: struct errors carry a type name, and `retry.on` filters which failures are retried.

## Highlights

- **Struct `__type` tagging** — `NetworkError(message: "timeout")` is identifiable at runtime
- **`retry.on=NetworkError,str`** — retry only matching error types
- **Type checking** — unknown struct names in `on=` fail at compile time (E2112)

## Install

```bash
go install github.com/jiejie-dev/funny/v2/cmd/funny@v2.1.4
```
